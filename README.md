# StockfishBlazorChess

StockfishBlazorChess is a chess application built with Blazor Server and MudBlazor, featuring gameplay powered by the Stockfish chess engine. This project offers a high-quality chess experience against a powerful AI opponent, with adjustable difficulty settings.
